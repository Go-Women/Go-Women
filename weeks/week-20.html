    This Week's Agenda
    1. Explanation on how the backend and frontend communicate
      1. GET
        • Used to retrieve data from the database.
        • A request is sent with a query, and data matching the query is returned from the database.
      2. POST
        • Used to create new data in the database.
        • A request is sent containing the data to be added to the database.
      3. PATCH
        • Used to update existing data in the database
        • A request is sent identifying the data to be replaced and what to replace it with.
      4. DELETE
        • Used to remove data from the database.
        • We will not be using this, but "hiding" data instead.
          • Deleting data is bad practice.
